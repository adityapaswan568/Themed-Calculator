# Themed Calculator

A simple, modern, and themeable calculator built with HTML, CSS, and JavaScript.

## Features

- **Basic Arithmetic Operations:** Perform addition, subtraction, multiplication, and division.
- **Percentage Calculation:** Easily calculate percentages.
- **Themeable:** Change the theme color of the calculator using a color picker. The background of the page also changes to a lighter shade of the selected color.
- **Responsive Design:** The calculator is responsive and works on different screen sizes.
- **Modern UI:** A clean and modern user interface with a glass-like effect and smooth animations.
- **Clear and Delete:** Clear the screen or delete the last character with dedicated buttons.

## How to Use

1.  Clone the repository or download the files.
2.  Open the `index.html` file in your web browser.
3.  Use the calculator buttons to perform calculations.
4.  Click on the color picker at the bottom right to change the theme color.

## File Structure

```
.
├── index.html
├── script.js
└── style.css
```

## Code Overview

-   **`index.html`:** The main HTML file that contains the structure of the calculator.
-   **`style.css`:** The CSS file that styles the calculator and provides the themeing functionality.
-   **`script.js`:** The JavaScript file that handles the calculator's logic, including the arithmetic operations and theme changes.
